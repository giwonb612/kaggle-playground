🔗 스태킹 앙상블(Stacking Ensemble)
앙상블 & 스태킹이란?
앙상블(Ensemble): 여러 모델의 예측을 결합해 단일 모델보다 높은 성능 달성

스태킹(Stacking): 기본 모델들의 예측을 새로운 피처로 사용해 메타 모델을 학습
OOF란?
OOF (Out-of-Fold): 각 폴드에서 검증 예측만 사용
→ 모델이 학습하지 않은 데이터의 예측
→ 데이터 누수 완전 방지
스태킹 구조

LR RF GB XGB LGBM SVM
